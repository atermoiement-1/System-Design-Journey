# System Design Journey

## About

This repository documents my journey of learning System Design and understanding how large-scale software systems are built.

The goal is not only to learn how applications work but also to understand how they scale to millions of users.

---

## Why I Started This Repository

While building software projects, I realized that writing code alone is not enough.

A software engineer should also understand:

- System Architecture
- Databases
- APIs
- Scalability
- Performance
- Reliability

This repository will contain my notes, diagrams, and learnings.
